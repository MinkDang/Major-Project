  • MDN JS Guidelines
  • MDN HTML Guidelines

List of improvements
  • HTML
    • Matching MDN HTML Guidelines
    • Change sidebar container from span to a for native site linking
    • Update quantity keystroke block from event.keycode (deprecated) to event.key
    • Submit and Reset is now button instead of div
  • HTML & JS
    • Trigger events on input instead of label to increase accessibility
    • Move events to JS (remove all function call from HTML)
    • Remove Credit Card readOnly
    • Better regex validation
  • JS
    • Matching Airbnb and MDN Guidelines
    • ESLint used
    • Different time input styling for Chrome
    • More arrow function usage
    • More ternary operator whenever possible
    • Logistic Handling is shorten (20 lines to 15 lines)
    • Dictionaries are used to replace if else if possible
    • Intervals are global variables
    • Let enforceTimeRestrictions() handle time update in when user chooses "Now"
    • Submit button blurred when there are errors within the order
    • textarea scroll to bottom when appending order
    • input related errors in resetButton is fixed
  • CSS
    • Matching MDN CSS Guidelines
    • :focus-visible used to increase accessibility
    • Remove unnecessary vendor prefixes, rules
    • input display is not none, opacity: 0; margin: 0; is used instead to increase accessibility
    • Smoother animation

v2.07 focuses on changing how CSS works fundamentally. alert() is replaced by a pop up notification. Minor accessibility improvements are included.
